## 功能
- 🔍 **PR智能审查** — 自动分析代码变更、给出审查意见+评分
- 📋 **Issue自动分类** — 自动打标签、分析问题、给初步建议
- 🆓 **零成本** — 基于OpenRouter免费模型

## 5分钟安装
1. 去 [openrouter.ai](https://openrouter.ai) 免费注册，获取API Key
2. 仓库 Settings → Secrets → 添加 `OPENROUTER_KEY`
3. 复制 `.github/workflows/` 下的两个yml到你的仓库
4. 提交PR或创建Issue试试
